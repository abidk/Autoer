Autoer
======

Developed in 2007 to help automate specific actions in some games i.e. Runescape. Open sourcing the project for educational purposes.

Script the execution of repetitive keyboard and mouse actions.

You can script a number of actions:

1. Delay - Pause the execution
2. Mouse move - Move the mouse to a specific location
3. Mouse click - Click the mouse in a specific location
4. Check colour - Check the pixel colour where the mouse pointer is set. You can create sub actions if the colour matches. 
5. Key press - Press a keyboard key
